== Overview

ClubHub is a desktop address book application used for managing Co-Curricular Activities (CCAs). The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and has about 10 kLoC.

== Summary of contributions

* *Major enhancement*: added the Item List which stores entries of Items and their respective quantities and locations.
** What it does: allows the user to add entries of Items into the Item List. Entries can then be deleted and/or edited using their respective commands.
** Justification: This feature improves the product significantly because a user can manage Items in the Item List.
** Highlights:

* *Minor enhancement*: added the undoAll/redoAll commands that allow the user to undo/redo all changes made to ClubHub.
